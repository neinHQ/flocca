# Security Policy

## Supported Versions

Since Flocca is a cloud-connected VS Code extension, we only support the latest version available on the VS Code Marketplace. We strongly recommend always keeping your extension auto-updated.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

We take the security of our users and their data very seriously.

**Do not open public issues for security vulnerabilities.**

If you believe you have found a security vulnerability in Flocca, please report it to us as follows:

1.  **Email**: Send the details to **hello@flocca.app**.
2.  **Response**: We will acknowledge your report within 48 hours.
3.  **Process**: We will investigate the issue and provide an estimated timeline for a fix.

We ask that you do not disclose the vulnerability specifically to the public until we have had the opportunity to address it.

Thank you for helping keep Flocca safe!
