# Installation

## Prerequisites
- **VS Code**: Version 1.85.0 or higher.
- **Node.js**: Version 18+ (Required for running local MCP servers).
- **Git**: Required for GitHub agent git operations.

## Install from Marketplace
1. Open **Visual Studio Code**.
2. Go to the **Extensions** view (`Ctrl+Shift+X` or `Cmd+Shift+X`).
3. Search for `Flocca`.
4. Click **Install**.

## Post-Installation
After installing, you will see the Flocca icon (Terminal symbol) in the Activity Bar.
1. Click the **Flocca** icon to open the Dashboard.
2. Use the **Connect** buttons to set up your agents.
3. Open the **Command Palette** (`Cmd+Shift+P`) and type `Flocca` to see available commands.
