# Jira Agent

## Overview
The Jira agent enables seamless project management within VS Code.

## Features
- **Search Issues**: Find issues by JQL or ID.
- **View Details**: Read comments, status, and assignees.
- **Transitions**: Update issue status.

## Setup
1. **Get a Token**:
   - Go to [Atlassian Account Settings](https://id.atlassian.com/manage-profile/security/api-tokens).
   - Create an API Token.
2. **Connect**:
   - Click **Connect** in Flocca.
   - Provide your **Jira Site URL** (e.g., `https://domain.atlassian.net`).
   - Enter your Email and API Token.

## Usage
- Command: `Flocca: Jira Shell` to run JQL.
- Chat: *"Show me tickets assigned to me"*
