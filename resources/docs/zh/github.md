# GitHub 代理

## 概览
 GitHub 代理允许您直接从 VS Code 访问您的代码库。您可以搜索代码库、管理 Issue 并读取文件。

## 功能
- **搜索代码库**
- **读取文件**
- **Git 操作**: `add`, `commit`, `push`
- **Pull Request**: 创建与合并 PR
- **Issue 管理**

## 设置
1. **获取 Token**:
   - 前往 [GitHub Developer Settings](https://github.com/settings/tokens).
   - 生成 **Personal Access Token (Classic)**.
   - 选择权限: `repo`, `user`, `read:org`.
2. **连接**:
   - 点击 Flocca 中的 **Connect** 按钮。
   - 输入 Token。
